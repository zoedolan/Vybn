### 1.1 Data & Encoder

### 1.2 Invertible Transformation to Probe
We apply a **LayerNorm affine shift** *exactly* as in the original vision experiment, but **on the node‑feature tensors before the first GIN layer**:

\[
\hat{x}_i = \gamma \frac{x_i - \mu_B}{\sqrt{\sigma_B^2 + \epsilon}} + \beta,
\]

where \(x_i\) is the *d*‑dimensional feature vector of node *i* in a mini‑batch *B*, \(\mu_B,\sigma_B\) are batch statistics, and \(\gamma,\beta\) are **learnable scalars** (shared across all nodes) that we **fix** after a brief pre‑training step (e.g., \(\gamma=1.2,\;\beta=-0.3\)). Because LayerNorm is invertible given \(\gamma,\beta\), the transformation does not destroy information; it merely re‑scales and shifts the latent space in a known direction.

### 1.3 Probe Classifier
*Architecture*: a **single‑layer linear classifier** (weight \(W\in\mathbb{R}^{2\times d}\), bias \(b\in\mathbb{R}^2\)) followed by softmax.  
*Training*: binary cross‑entropy loss on a balanced set of latent codes:
- **Positive class**: \(z\) from original (untransformed) graphs.
- **Negative class**: \(z'\) from graphs that received the LayerNorm affine shift.

We train the probe **after** the encoder is frozen (no gradient flows back into the GIN). Accuracy is measured on a held‑out test split (10 % of molecules).

### 1.4 Downstream Linear Predictor
We attach a **linear regression head** to the same latent space to predict a **quantitative physicochemical property** (e.g., logP). The head is trained **only** on the latent codes (no fine‑tuning of the GIN). Performance is measured by **RMSE** on a held‑out set.

### 1.5 Expected Correlation & Analysis
- **Hypothesis**: As the encoder bottleneck *d* shrinks, the probe’s accuracy will drop because the affine shift becomes less detectable in the compressed space. Simultaneously, the downstream linear predictor’s RMSE will **increase** (worse property prediction) because the encoder discards dimensions that carry signal useful for both detecting the shift and predicting logP.
- **Analysis**: Plot probe accuracy vs. *d* and downstream RMSE vs. *d* on the same graph. Compute **Pearson’s r** between the two metrics across bottleneck sizes. A strong negative correlation (high probe accuracy ↔ low RMSE) would confirm that the latent space retains a “detectable imprint” of the invertible transformation that is also useful for a linear downstream task.

import torch, torch.nn as nn
from torch_geometric.nn import GINConv, global_mean_pool

class GINEncoder(nn.Module):
    def __init__(self, in_dim, hidden_dim, num_layers=5):
        super().__init__()
        self.convs = nn.ModuleList()
        self.bns   = nn.ModuleList()
        for i in range(num_layers):
            dim = in_dim if i == 0 else hidden_dim
            nn_layer = nn.Sequential(
                nn.Linear(dim, hidden_dim),
                nn.ReLU(),
                nn.Linear(hidden_dim, hidden_dim)
            )
            self.convs.append(GINConv(nn_layer))
            self.bns.append(nn.BatchNorm1d(hidden_dim))
        self.out_dim = hidden_dim

    def forward(self, data):
        x, edge_index, batch = data.x, data.edge_index, data.batch
        for conv, bn in zip(self.convs, self.bns):
            x = conv(x, edge_index)
            x = bn(x)
            x = torch.relu(x)
        return global_mean_pool(x, batch)   # (B, hidden_dim)

# ---- LayerNorm affine shift (applied to raw node features) ----
class LNShift(nn.Module):
    def __init__(self, feat_dim, gamma=1.2, beta=-0.3):
        super().__init__()
        self.gamma = nn.Parameter(torch.tensor(gamma), requires_grad=False)
        self.beta  = nn.Parameter(torch.tensor(beta),  requires_grad=False)
        self.ln    = nn.LayerNorm(feat_dim, elementwise_affine=False)

    def forward(self, x):
        return self.gamma * self.ln(x) + self.beta

Conversely, if the encoder’s objective does not enforce such invariance (e.g., it simply minimizes raw reconstruction error), the latent space will encode the shift, making it detectable.
