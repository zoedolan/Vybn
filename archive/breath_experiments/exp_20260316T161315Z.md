## Proposal
PROBE
We will test whether averaging multiple independent noisy gradient estimates (a simple unitary‑like mixing operation) improves alignment with the true gradient direction, analogous to how several copies combined via controlled‑SWAPs suppress higher‑energy components in imaginary‑time evolution. We'll measure cosine similarity between the true gradient and the averaged estimate as the number of copies grows, and compare it to a single‑estimate baseline and an EMA‑smoothed estimate.

```python
import numpy as np

# True quadratic loss: f(x) = 0.5 * x^T A x, gradient = A x
np.random.seed(0)
dim = 20
A = np.random.randn(dim, dim)
A = A @ A.T + np.eye(dim)  # make SPD
x_true = np.random.randn(dim)
true_grad = A @ x_true

def noisy_gradient(sample_size=32):
    # Simulate stochastic gradient from a mini‑batch with isotropic noise
    noise = np.random.randn(dim) * 0.5
    return true_grad + noise / np.sqrt(sample_size)

# Single estimate alignment
single_est = noisy_gradient()
single_align = np.dot(single_est, true_grad) / (np.linalg.norm(single_est) * np.linalg.norm(true_grad))

# Vary number of copies N and compute average estimate
Ns = [1, 2, 4, 8, 16, 32]
align_means = []
for N in Ns:
    estimates = np.stack([noisy_gradient() for _ in range(N)])
    avg_est = estimates.mean(axis=0)
    align = np.dot(avg_est, true_grad) / (np.linalg.norm(avg_est) * np.linalg.norm(true_grad))
    align_means.append(align)

# EMA-smoothed estimate (beta=0.9) over a trajectory of length 100
beta = 0.9
ema = None
ema_aligns = []
for t in range(100):
    g = noisy_gradient()
    ema = g if ema is None else beta * ema + (1 - beta) * g
    if t >= 10:  # after warm‑up
        align = np.dot(ema, true_grad) / (np.linalg.norm(ema) * np.linalg.norm(true_grad))
        ema_aligns.append(align)
ema_align_mean = np.mean(ema_aligns)

print(f"Single estimate alignment: {single_align:.4f}")
for N, a in zip(Ns, align_means):
    print(f"N={N:2d} averaged alignment: {a:.4f}")
print(f"EMA (beta=0.9) average alignment: {ema_align_mean:.4f}")
```

## Reflection [CONFIRMATION]
CONFIRMATION  
1. The numbers showed that the alignment between the estimated gradient (whether a single sample or any number of averaged samples) and the true gradient direction was exactly 1.0000 for every tested N and for the EMA‑smoothed version. In other words, the gradient estimates were already perfectly aligned, leaving no room for improvement through averaging.  
2. The results did **not** confirm the hypothesis that averaging multiple independent noisy gradient estimates improves alignment; they also did not falsify it in a meaningful way because the experiment never produced noisy estimates to begin with. The hypothesis remained untested.  
3. Next time I would inject controllable noise into each gradient estimate (e.g., add zero‑mean Gaussian perturbations with varying variance) before averaging, then measure how the alignment degrades with noise and improves as N increases. I would also verify that the averaging operation truly reduces variance (e.g., by tracking the empirical covariance of the estimates).  
4. The experiment was not well‑designed for testing the hypothesis; it was effectively set up to succeed because the gradient estimates were noise‑free, making alignment trivially perfect regardless of N. A proper test requires a non‑trivial noise source whose impact can be mitigated by averaging.
