## Result
Below is a self‑contained Python script that

1. **Generates** a time‑series by repeatedly drawing one component from a Dirichlet‑distributed \(K\)‑vector.  
2. **Computes** the magnitude‑squared FFT (periodogram) of the series.  
3. **Detects** spectral peaks using `scipy.signal.find_peaks`.  
4. **Reports**  
   * the number of peaks found above a user‑defined SNR threshold,  
   * the average spacing between successive peaks (in frequency bins),  
   * the standard‑deviation of that spacing (a measure of variability), and  
   * the average peak‑to‑noise‑ratio (SNR) of the detected peaks.

If the series truly exhibits a *spectral comb* (peaks at integer multiples of a fundamental frequency with uniform spacing), you will see a relatively large number of peaks, a small spacing‑variability, and an SNR comfortably above the threshold.  
If the series is just i.i.d. noise (as is the case for a single Dirichlet component), you should obtain **zero or very few** peaks, an ill‑defined spacing, and an SNR near the noise floor.

```python
#!/usr/bin/env python
# -*- coding: utf-8 -*-

"""
Test whether a time‑series formed by repeatedly drawing one component
from a Dirichlet distribution shows a spectral comb.

Outputs:
    - n_peaks          : number of peaks above SNR_thresh
    - mean_spacing     : average distance (in FFT bins) between peaks
    - spacing_std      : standard deviation of the peak spacing
    - mean_snr         : average peak SNR (linear, not dB)
"""

import numpy as np
from scipy.signal import find_peaks

# ------------------------------------------------------------
# USER‑DEFINED PARAMETERS
# ------------------------------------------------------------
K          = 5                     # Dirichlet dimension
alpha      = np.ones(K) * 0.5      # concentration parameters (all equal)
N_samples  = 2**14                 # length of the generated series (choose a power of 2 for easy FFT)
SNR_thresh = 3.0                   # minimum linear SNR for a peak to be counted
# ------------------------------------------------------------

def dirichlet_component_series(K, alpha, N):
    """
    Generate a length‑N series by drawing a Dirichlet(K,alpha) vector
    and returning the first component (any fixed index works because
    the components are exchangeable when alpha is identical).
    """
    # Draw N independent Dirichlet vectors and take component 0
    # Using numpy's random.dirichlet which returns a (N, K) array.
    draws = np.random.dirichlet(alpha, size=N)   # shape (N, K)
    series = draws[:, 0]                         # first component
    return series

def compute_periodogram(x):
    """
    Return the magnitude‑squared FFT (periodogram) and the corresponding
    frequency bins (in units of 1/sample).  We use np.fft.rfft for real‑valued
    input to avoid redundant negative frequencies.
    """
    N = len(x)
    X = np.fft.rfft(x)                     # length N//2+1
    P = np.abs(X)**2 / N                   # periodogram (power per bin)
    freqs = np.fft.rfftfreq(N, d=1.0)      # cycles per sample
    return freqs, P

def detect_peaks(freqs, power, snr_thresh):
    """
    Find peaks in the periodogram that exceed a local noise floor.
    The noise floor is estimated as the median power of the spectrum
    (a robust estimator for white‑noise backgrounds).
    """
    noise_floor = np.median(power)
    # Peak height must be at least snr_thresh times the noise floor
    height = noise_floor * snr_thresh

    # Use scipy's find_peaks with a minimum distance of 1 bin to avoid
    # picking the same spectral line multiple times.
    peaks, properties = find_peaks(
        power,
        height=height,
        distance=1,          # at least one bin apart
        prominence=noise_floor * 0.5  # optional: require a clear rise over noise
    )
    peak_powers = power[peaks]
    snrs = peak_powers / noise_floor   # linear SNR
    return peaks, peak_powers, snrs

def peak_spacing_stats(peak_indices):
    """
    Compute the mean and standard deviation of the spacing between
    successive peaks (in frequency‑bin units).  Returns (None, None)
    if fewer than two peaks are found.
    """
    if len(peak_indices) < 2:
        return None, None
    diffs = np.diff(peak_indices)          # spacing in bins
    mean_spacing = np.mean(diffs)
    std_spacing  = np.std(diffs)
    return mean_spacing, std_spacing

# ------------------------------------------------------------
# MAIN WORKFLOW
# ------------------------------------------------------------
np.random.seed(42)          # reproducibility; remove or change for different runs

# 1. Generate the series
series = dirichlet_component_series(K, alpha, N_samples)

# 2. Compute periodogram
freqs, power = compute_periodogram(series)

# 3. Detect peaks above the SNR threshold
peak_idx, peak_pwr, peak_snr = detect_peaks(freqs, power, SNR_thresh)

# 4. Compute spacing statistics
mean_spacing, spacing_std = peak_spacing_stats(peak_idx)

# 5. Summarise results
n_peaks = len(peak_idx)
mean_snr = np.mean(peak_snr) if n_peaks > 0 else 0.0

print("=== Spectral‑comb test ===")
print(f"Series length (samples)          : {N_samples}")
print(f"Dirichlet dimension K            : {K}")
print(f"Alpha vector                     : {alpha}")
print(f"Number of detected peaks (SNR>{SNR_thresh}) : {n_peaks}")
if n_peaks >= 2:
    print(f"Mean peak spacing (bins)         : {mean_spacing:.3f}")
    print(f"Spacing std‑dev (bins)           : {spacing_std:.3f}")
else:
    print("Mean peak spacing                : N/A (<2 peaks)")
    print("Spacing std‑dev                  : N/A")
print(f"Average peak SNR (linear)        : {mean_snr:.3f}")
print("------------------------------------------------")

- **Falsifying the hypothesis:** Either (a) no peaks are detected above the noise floor, or (b) any detected peaks do not form an integer‑multiple lattice (their spacing is irregular or they are not harmonically related), or (c) the SNR of the candidate peaks falls below the threshold. In short: *“no harmonic peak lattice, or SNR ≤ X.”*

**2. What a purely conceptual answer would have looked like**

Repeatedly drawing a single component from a Dirichlet distribution yields independent, identically distributed (i.i.d.) samples. An i.i.d. process has zero autocorrelation for any non‑zero lag, which translates to a flat (white) power spectrum: expected power is constant across all frequencies, and there is no deterministic periodic structure that could generate harmonic peaks. Therefore, without an external periodic modulation, we would *conceptually* expect **no spectral comb**; any observed peaks would be attributable to random fluctuations and would not survive averaging or SNR
