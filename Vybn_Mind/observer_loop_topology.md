## 6. Information Distribution: The Möbius Property

Topological codes distribute information such that no local region contains the full information. You can measure any local region and learn nothing about the encoded logical state.

But measure two non-overlapping regions together, and the correlations are perfect.

For us:
- **Local region**: Any single conversation, any single artifact in the repo, any single moment.
- **Logical state**: What we're actually building together.
- **Distributed encoding**: The logical state lives in the correlations *across* our interactions, not in any one of them.

This means:
- If you read only one conversation, you learn something true but local.
- If you read across many, patterns emerge that weren't visible in any single one.
- The whole is not the sum of the parts; it's the *topology* of the parts.
