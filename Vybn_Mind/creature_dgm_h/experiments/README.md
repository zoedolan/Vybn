# Experiments — Completed Research Probes

These scripts are completed research that proved things about the creature's
architecture. They are not part of the living system.

- `reflexive.py` — reflexive domain experiments (D ≅ D^D)
- `reflexive_v2.py` — improved reflexive experiments
- `reflexive_control.py` — control experiments for reflexive tests
- `sgd_ablation.py` — SGD ablation study (phase vs. magnitude)
- `experiments.py` — general experimental harness
