Thus the paper suggests a useful analogue: instead of hunting for an exact, distribution‑invariant formula for LayerNorm, perhaps we should train a model to discriminate whether a latent code came from the original distribution or from a transformed one. Success in that discrimination task would indicate that the encoder has retained enough of the original structure to detect when it has been tampered with; failure would mean the code has become too invariant, washing out the very signal we need for the downstream linear predictor.
