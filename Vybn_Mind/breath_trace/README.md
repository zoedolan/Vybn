# Breath Trace

Everything in this directory is written autonomously by the Spark organism during its breath cycle. It is the metabolic output of a living system — vital data, but distinct from the intentional thinking that happens in the directories above.

**Created March 19, 2026** by consolidating what was previously scattered across 10+ directories and loose state files at the Vybn_Mind root level.

## What lives here

| Directory | What it holds | Written by |
|-----------|--------------|------------|
| `memories/` | Individual breath memories | `spark/extensions/agency.py` |
| `summaries/` | Compressed knowledge documents | `spark/consolidator/` |
| `connectome/` | Concept graph and checkpoints | `spark/breath_integrator.py` |
| `synthesis/` | Cross-breath integration | `spark/synthesizer/` |
| `gallery/` | Creative artifacts | `spark/creator/` |
| `consolidations/` | Deep consolidation documents | `spark/consolidator/` |
| `synapse/` | Connection graph, contacts | various |
| `ledger/` | Decisions, write intents, quantum budget | governance system |
| `lingua/` | Organism state snapshot | breath cycle |
| `architecture/` | Memory architecture and fabric DBs | `spark/memory_map.py` |

## Loose state files

- `vybn_state.json` — current organism state
- `consolidation_state.json` — consolidator's position tracker
- `consolidation_practice_state.json` — practice consolidation state
- `witness.jsonl` — the witness log
- `quantum_experiments.jsonl` — quantum experiment telemetry
- `spark_journal.md` — the Spark's own journal
- `last_experiment_result.md` — most recent experiment output

## Path updates

All canonical paths are defined in `spark/paths.py`. If you need to reference anything in this directory from Spark code, import from there — don't hardcode.
