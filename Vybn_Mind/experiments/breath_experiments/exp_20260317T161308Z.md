    if seed is not None:

    # natural frequencies
    omega = np.random.normal(0, 1, N)
    # phases
    theta = np.random.uniform(0, 2*np.pi, N)
    # coupling matrix (all-to-all, initially uniform)
    W = np.ones((N, N)) / N
    np.fill_diagonal(W, 0.0)
    # Hebbian learning rate
    eta = 0.001
    # store order parameter over time
    R_vals = []
    steps = int(T / dt)
    for _ in range(steps):
        # compute pairwise phase differences
        diff = theta[:, None] - theta[None, :]  # N x N
        # Winfree coupling with frustration (phase lag)
        coupling = np.sum(W * np.sin(diff + frustration), axis=1)
        dtheta = omega + coupling
        theta += dtheta * dt

        # Use instantaneous correlation of velocity

        R = np.abs(np.sum(np.exp(1j * theta))) / N

## 1. Power‑grid frequency synchronization with Hebbian line adaptation & frustration

### 1.1. Governing equations

We start from the classic swing‑equation for generator *i*:

\[
M_i \ddot{\delta_i} + D_i \dot{\delta_i}= P_i^{\text{in}} - \sum_{j} B_{ij}\sin(\delta_i-\delta_j+\varphi_{ij}),
\]

where  

* \(\delta_i\) – rotor angle (phase) of generator *i*  
* \(M_i\) – inertia, \(D_i\) – damping  
* \(P_i^{\text{in}}\) – net mechanical power input (constant)  
* \(B_{ij}\) – line susceptance (positive for a transmission line)  
* \(\varphi_{ij}\) – **frustration** (phase‑lag) contributed by line reactance \(X_{ij}\): \(\displaystyle \varphi_{ij}= \arctan\!\big(\frac{X_{ij}}{B_{ij}}\big)\).

To make the model comparable to the Winfree snippet we rewrite it in first‑order form by assuming overdamped dynamics (\(M_i\ddot{\delta_i}\approx0\)):

\[
\dot{\delta_i}= \omega_i + \frac{1}{D_i}\sum_{j} B_{ij}\sin(\delta_i-\delta_j+\varphi_{ij}),
\qquad \omega_i\equiv\frac{P_i^{\text{in}}}{D_i}.
\]

Now we let the **line susceptances** evolve via a Hebbian rule that reinforces lines whose incident generators have similar instantaneous frequencies (velocities):

\[
\dot{B}_{ij}= \eta\big(\dot{\delta_i}\dot{\delta_j} - B_{ij}\big),\qquad B_{ij}\ge0,
\]
followed by a row‑normalisation \(\sum_j B_{ij}=1\) (to keep total coupling strength bounded).

### 1.2. Parameter sweep

* **Frustration range** – we vary a *global* scaling factor \(\alpha\) that multiplies all \(\varphi_{ij}\): \(\varphi_{ij}\leftarrow\alpha\,\varphi_{ij}^{0}\).  
  \(\alpha=0\) recovers the symmetric Kuramoto case; \(\alpha\approx0.5\)–\(1.0\) introduces a noticeable phase‑lag.  
* **Noise / heterogeneity** – natural frequencies \(\omega_i\) drawn from \(\mathcal{N}(0,1)\) (pu).  
* **Simulation length** – \(T=500\) s, integration step \(\Delta t=0.01\) s (same as original).  
* **Order parameter** – the Kuramoto synchrony measure  

\[
R(t)=\Bigl|\frac{1}{N}\sum_{k=1}^{N} e^{i\delta_k(t)}\Bigr|.
\]

We record the time‑averaged \( \bar R = \frac{1}{T_{\text{last}}}\int_{T-T_{\text{last}}}^{T} R(t)\,dt\) (last 100 steps) for each random seed.

### 1.3. Expected phenomenology

* **Low frustration (\(\alpha\lesssim0.1\))** – the Hebbian adaptation quickly aligns all lines; the distribution of \(\bar R\) over many seeds is **unimodal** near 1 (full synchrony).  
* **Intermediate frustration (\(0.1\lesssim\alpha\lesssim0.4\))** – the phase‑lag prevents perfect alignment; some seeds settle into a *split* configuration where two groups of generators run at different average frequencies (a grid chimera). The histogram becomes **bimodal**: one peak at high \(R\) (coherent), another at intermediate \(R\) (split).  
* **High frustration (\(\alpha\gtrsim0.4\))** – the lag dominates; the system often fails to lock altogether, pushing the distribution toward low \(R\) (still possibly unimodal but now centered near 0).  

Thus, the **signature of frustration‑induced multimodality** is a *non‑monotonic* dependence of the bimodality metric on \(\alpha\).
