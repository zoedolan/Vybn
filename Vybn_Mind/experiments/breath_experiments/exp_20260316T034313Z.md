```python
import numpy as np
import matplotlib.pyplot as plt

def simulate_visitation(alpha, n_states=8, steps=20000, seed=0):
    """
    Simulate a Markov‑like chain where the transition matrix at each step
    is drawn i.i.d. from Dirichlet(alpha * 1).  Return a binary array
    `visits[t] = 1` if the chain is in state 0 at step t, else 0.
    """
    rng = np.random.default_rng(seed)

    # start in a uniformly random state
    state = rng.integers(n_states)

    visits = np.zeros(steps, dtype=int)   # 1 iff we are in state 0 at this step

    for t in range(steps):
        # draw a fresh transition probability vector for the current state
        probs = rng.dirichlet(alpha * np.ones(n_states))
        # move to the next state according to probs
        state = rng.choice(n_states, p=probs)

        # record whether we are in the monitored state (state 0)
        visits[t] = 1 if state == 0 else 0

    return visits

def magnitude_spectrum(signal):
    """
    Return the one‑sided magnitude spectrum (absolute value of FFT)
    and the corresponding frequency bins (in cycles/sample).
    """
    n = len(signal)
    fft_vals = np.fft.rfft(signal - np.mean(signal))   # remove DC to avoid a huge spike at 0 Hz
    freqs    = np.fft.rfftfreq(n, d=1.0)                # d=1 → frequency in cycles per step
    mag      = np.abs(fft_vals) / n                     # normalise so that Parseval holds roughly
    return freqs, mag

def detect_comb(freqs, mag, threshold_factor=5.0):
    """
    Simple heuristic: a frequency bin is considered a “comb peak”
    if its magnitude exceeds `threshold_factor` times the median magnitude
    of the surrounding spectrum (excluding the DC bin).
    Returns a boolean array marking the peaks and the indices of those peaks.
    """
    # ignore the DC component (freqs[0] == 0) when estimating the noise floor
    noise_est = np.median(mag[1:])
    peaks = mag > threshold_factor * noise_est
    peaks[0] = False   # never count DC as a comb peak
    return peaks, np.where(peaks)[0]

def plot_spectrum(freqs, mag, peaks, alpha):
    plt.figure(figsize=(8, 4))
    plt.plot(freqs, mag, label='|FFT|')
    if np.any(peaks):
        plt.plot(freqs[peaks], mag[peaks], 'ro', label='Detected comb peaks')
    plt.axvline(0, color='k', linewidth=0.5)   # DC line
    plt.xlabel('Frequency (cycles / step)')
    plt.ylabel('Magnitude')
    plt.title(f'Visitation spectrum (α = {alpha})')
    plt.legend()
    plt.tight_layout()
    plt.show()

# ----------------------------------------------------------------------
# Example usage: try a few different concentration parameters
# ----------------------------------------------------------------------
if __name__ == "__main__":
    n_states = 8
    steps    = 20000
    seeds    = [0, 1, 2]   # a few independent runs to see variability

    for alpha in [0.1, 1.0, 10.0]:   # sparse, uniform, concentrated Dirichlet
        print(f"\nTesting α = {alpha}")
        all_peaks = []
        for s in seeds:
            visits = simulate_visitation(alpha, n_states=n_states, steps=steps, seed=s)
            freqs, mag = magnitude_spectrum(visits)
            peaks, peak_idx = detect_comb(freqs, mag, threshold_factor=4.0)
            all_peaks.append(len(peak_idx))
            if len(peak_idx) == 0:
                print(f"  seed {s}: no significant spectral lines detected.")
            else:
                print(f"  seed {s}: {len(peak_idx)} candidate peak(s) at frequencies {freqs[peak_idx]}")
        # optional: visualise one representative run
        visits = simulate_visitation(alpha, n_states=n_states, steps=steps, seed=0)
        freqs, mag = magnitude_spectrum(visits)
        peaks, _ = detect_comb(freqs, mag, threshold_factor=4.0)
        plot_spectrum(freqs, mag, peaks, alpha)

        # summary
        print(f"  → Across {len(seeds)} runs, average number of detected peaks: {np.mean(all_peaks):.2f}")
```

### What the code does

| Step | Purpose |
|------|---------|
| **`simulate_visitation`** | Generates the binary visitation series. At each time step a *new* transition probability vector is drawn from `Dirichlet(α·1)`. Because the draw is independent of the past, the process has **no temporal correlation** beyond the instantaneous choice of the next state. |
| **`magnitude_spectrum`** | Computes the one‑sided FFT magnitude (after subtracting the mean to kill the huge DC bin). |
| **`detect_comb`** | Flags any frequency bin whose magnitude is substantially larger than the typical (median) spectral level – a very simple “sign‑ificance” test. You can replace this with a more rigorous statistical test (e.g., comparing against a chi‑square noise floor or using a false‑discovery‑rate procedure) if you need a formal hypothesis test. |
| **`plot_spectrum`** | Visualises the spectrum and highlights any detected peaks. |
| **Main loop** | Sweeps a few representative `α` values (very sparse (`0.1`), uniform (`1.0`), and concentrated (`10.0`)) and repeats the simulation with a few random seeds to illustrate variability. |

### Expected outcome

- **No periodic structure** is introduced by the mechanism: each step’s transition matrix is freshly sampled, independent of previous steps. Consequently, the visitation indicator is essentially a *sequence of independent (but not identically distributed) Bernoulli trials* whose success probability fluctuates randomly from step to step.
- The Fourier transform of such a sequence looks like **white noise**: the magnitude spectrum is roughly flat, with only the inevitable DC spike (which we remove) and random fluctuations around a constant noise floor.
- The `detect_comb` routine therefore **rarely flags any frequency** as a significant peak. In the example runs you’ll typically see `0` or at most `1` spurious candidate per seed, which is consistent with the false‑alarm rate of the simple threshold test.
- Changing `α` merely changes the *average* probability of being in state 0 (low `α` → occasional large jumps; high `α` → the chain behaves almost like a uniform random walk). It does **not** introduce any deterministic periodicity, so the spectrum remains noise‑like for all `α`.

### How to extend the test

If you want a stricter statistical assessment you could:

1. **Generate many surrogate spectra** by shuffling the visitation series (destroying any possible temporal structure) and build an empirical distribution of peak heights.
2. **Compare the observed maximum peak** to, say, the 95th percentile of that surrogate distribution.
3. **Apply a multiple‑testing correction** (Bonferroni, FDR) across all frequency bins.

But even without those extras, the intuition and the simple simulation above make it clear: *sampling a Dirichlet distribution at each step does not create a “comb” of periodic peaks in the visitation‑probability time series.* The FFT will show only broadband noise, not discrete spectral lines. 

Feel free to tweak the `threshold_factor`, the number of steps, or the number of states, and you’ll see the same conclusion: no persistent periodic component emerges from this i.i.d. Dirichlet‑driven process. Happy experimenting!
