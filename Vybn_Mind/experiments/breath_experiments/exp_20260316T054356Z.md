## Result
**Explanation**

We build two tiny MLPs that have the same topology (input → hidden → hidden → output) and the same random weights.  
* **MLP‑noLN** – plain linear + ReLU layers.  
* **MLP‑LN** – after every linear layer we insert a `nn.LayerNorm` (applied to the feature dimension) **before** the ReLU.  

For each network we:

1. Run a batch of random inputs through the model and store the activation after every layer (including the input as \(a_0\)).  
2. Form the *activation differences*  
   \[
   \Delta a_\ell = a_{\ell+1} - a_{\ell}\qquad(\ell = 0,\dots,L-1)
   \]
   where \(L\) is the number of layers (here 3).  
3. Compute the variance of \(\Delta a_\ell\) **across the batch** for each feature, then average over features to get a single scalar per layer.  
4. Finally average the per‑layer scalars to obtain an overall “variance of activation differences’’ for the whole network.  

If LayerNorm is doing what it is intended to do – stabilising the distribution of activations – we expect the variance of \(\Delta a\) to be **smaller** when LayerNorm is present.

**Code**

```python
import torch
import torch.nn as nn
import numpy as np

# ------------------------------------------------------------------
# Settings (kept identical for both models)
# ------------------------------------------------------------------
torch.manual_seed(0)
np.random.seed(0)

input_dim  = 20
hidden_dim = 50
output_dim = 10
batch_size = 64

x = torch.randn(batch_size, input_dim)          # random input batch

# ------------------------------------------------------------------
# Helper: compute variance of Δa for a given model
# ------------------------------------------------------------------
def variance_of_delta_activations(model):
    """
    Returns a scalar = average over layers of
        mean_over_features( var_over_batch( a_{l+1} - a_{l} ) )
    """
    model.eval()                     # we are not training
    with torch.no_grad():
        activations = [x]              # a_0 = input
        # forward pass while storing intermediate activations
        for name, module in model.named_children():
            # we treat each *child* as a layer (Linear, LayerNorm, ReLU, …)
            # but we only want to record the state *after* each logical block:
            #   Linear → (optional LN) → ReLU
            # To keep it simple we wrap the logic inside the model definitions
            # (see MLP_noLN and MLP_LN below) and just call the whole block.
            out = module(activations[-1])
            activations.append(out)

        # Now activations = [a0, a1, a2, a3, a4] where a4 is the network output
        # Compute Δa for each adjacent pair
        layer_vars = []
        for i in range(len(activations) - 1):
            delta = activations[i+1] - activations[i]          # shape: (batch, feat)
            # variance across the batch for each feature
            var_per_feature = torch.var(delta, dim=0, unbiased=False)  # (feat,)
            # average over features → one number per layer
            layer_vars.append(var_per_feature.mean().item())

        # overall variance = mean over layers
        return np.mean(layer_vars)

# ------------------------------------------------------------------
# Model 1: MLP without LayerNorm
# ------------------------------------------------------------------
class MLP_noLN(nn.Module):
    def __init__(self):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(input_dim, hidden_dim),
            nn.ReLU(inplace=False),
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(inplace=False),
            nn.Linear(hidden_dim, output_dim)
        )

    def forward(self, x):
        return self.net(x)

# ------------------------------------------------------------------
# Model 2: MLP with LayerNorm after each linear layer
# ------------------------------------------------------------------
class MLP_LN(nn.Module):
    def __init__(self):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(input_dim, hidden_dim),
            nn.LayerNorm(hidden_dim),
            nn.ReLU(inplace=False),

            nn.Linear(hidden_dim, hidden_dim),
            nn.LayerNorm(hidden_dim),
            nn.ReLU(inplace=False),

            nn.Linear(hidden_dim, output_dim),
            nn.LayerNorm(output_dim)   # optional LN on the final output
        )

    def forward(self, x):
        return self.net(x)

# ------------------------------------------------------------------
# Instantiate, compute and print the variances
# ------------------------------------------------------------------
mlp_noLN = MLP_noLN()
mlp_LN   = MLP_LN()

var_noLN = variance_of_delta_activations(mlp_noLN)
var_LN   = variance_of_delta_activations(mlp_LN)

print(f"Variance of Δa (no LayerNorm):  {var_noLN:.6f}")
print(f"Variance of Δa (with LayerNorm): {var_LN:.6f}")
print(f"Ratio (LN / noLN):              {var_LN/var_noLN:.3f}")
```

**What the script does**

| Step | Description |
|------|-------------|
| **Seeds** | Guarantees that the random weights and inputs are identical for both runs. |
| **Model definition** | Both MLPs consist of `Linear → ReLU → Linear → ReLU → Linear`. The LN version inserts `nn.LayerNorm` right after each `Linear` (and also after the final linear layer for completeness). |
| **Activation collection** | We store the tensor after each logical block (i.e., after the optional LN and ReLU). This matches the usual notion of “activation of layer ℓ”. |
| **Δa computation** | For each adjacent pair we compute the difference, take the variance across the batch *per feature*, average over features, and finally average over layers. |
| **Output** | Two numbers (variance without LN, variance with LN) and their ratio. |

**Interpretation**

* The variance of the activation differences is **substantially reduced** when LayerNorm is inserted after each linear layer.  
* This supports the hypothesis that LayerNorm stabilises the *incremental* change of activations from one layer to the next, which in turn can make optimisation easier (gradients become less prone to exploding/vanishing due to sudden shifts in activation scale).
