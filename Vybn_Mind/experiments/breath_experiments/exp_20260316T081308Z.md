2. **What a purely conceptual answer could have looked like**  
   LayerNorm normalizes each feature to zero mean and unit variance **across the feature dimension**, not across the batch. Consequently, it rescales the *distribution of features* for each individual example, but it does not directly constrain how those features change from one example to another. Therefore, unless the activation changes happen to align with the feature dimensions that LayerNorm rescales, there is no reason to expect a systematic reduction in the batch‑wise variance of Δa.
