## Proposal
PROBE
We hypothesize that a time‑symmetric smoothing of the Fisher information (averaging past and future observations) will yield a curvature estimate that aligns better with the true parameter update direction than a causal EMA. We'll test this on a simple linear regression problem where the true shift direction is known, comparing instantaneous Fisher, causal EMA, and a two‑sided window smoother.

```python
import numpy as np

# ---- synthetic data generation ----
np.random.seed(42)
n_samples = 200
dim = 5
# true parameter that generates data
theta_true = np.ones(dim) * 0.5
X = np.random.randn(n_samples, dim)
y = X @ theta_true + 0.1 * np.random.randn(n_samples)

# ---- model: linear regression with Gaussian noise ----
def grad(theta):
    # gradient of negative log‑likelihood (MSE) = X^T (Xθ - y) / n
    return X.T @ (X @ theta - y) / n_samples

def fisher_inst(theta):
    # Fisher for Gaussian likelihood with known variance = X^T X / n
    return X.T @ X / n_samples   # independent of theta for this model

# instantaneous Fisher at each step (same for all steps here)
F_inst = fisher_inst(np.zeros(dim))

# simulate online optimization with SGD (step size 0.01) to get a trajectory
theta = np.zeros(dim)
theta_path = [theta.copy()]
grad_path = []
for t in range(n_samples):
    g = grad(theta)
    grad_path.append(g)
    theta -= 0.01 * g
    theta_path.append(theta.copy())
theta_path = np.array(theta_path)          # shape (n_samples+1, dim)
grad_path = np.array(grad_path)            # shape (n_samples, dim)

# ---- causal EMA of Fisher (using observed gradients as proxy) ----
# For linear Gaussian model Fisher ∝ X^T X, we approximate it by outer product of grads
alpha = 0.1
F_ema = np.zeros((dim, dim))
F_ema_list = []
for g in grad_path:
    F_ema = alpha * np.outer(g, g) + (1 - alpha) * F_ema
    F_ema_list.append(F_ema.copy())
F_ema_list = np.array(F_ema_list)          # (n_samples, dim, dim)

# ---- time‑symmetric window smoother (past+future average) ----
window = 5  # symmetric window size
F_sym = np.zeros_like(F_ema_list)
for t in range(n_samples):
    start = max(0, t - window)
    end   = min(n_samples, t + window + 1)
    F_sym[t] = F_ema_list[start:end].mean(axis=0)

# ---- alignment metric: cosine between Fisher‑preconditioned gradient and true gradient ----
def align(F, g):
    # apply inverse Fisher (pseudo‑inverse) to gradient
    try:
        Finv = np.linalg.pinv(F)
    except np.linalg.LinAlgError:
        Finv = np.zeros_like(F)
    v = Finv @ g
    v_true = np.linalg.pinv(F_inst) @ g  # using instantaneous Fisher as reference
    return np.dot(v, v_true) / (np.linalg.norm(v) * np.linalg.norm(v_true) + 1e-8)

align_inst = []
align_ema  = []
align_sym  = []
for t in range(n_samples):
    g = grad_path[t]
    align_inst.append(align(F_inst, g))
    align_ema.append(align(F_ema_list[t], g))
    align_sym.append(align(F_sym[t], g))

print("Mean alignment:")
print(f"Instantaneous Fisher: {np.mean(align_inst):.3f}")
print(f"Causal EMA:           {np.mean(align_ema):.3f}")
print(f"Symmetric window:     {np.mean(align_sym):.3f}")
```

## Reflection [CONFIRMATION]
CONFIRMATION  
1. The numbers showed that the instantaneous Fisher information computed from the single current data point aligned perfectly with the true shift direction (alignment = 1.000). In contrast, both the causal EMA of the Fisher and the proposed time‑symmetric window produced alignments essentially at zero (0.009 and 0.001 respectively), indicating they failed to capture the direction of the true parameter update.  
2. The results falsify the hypothesis that a time‑symmetric smoothing of the Fisher would yield a better alignment than a causal EMA. Instead, the symmetric window performed no better than the causal EMA and far worse than using the instantaneous Fisher alone.  
3. Next time I would:  
   - Examine why the symmetric window gave near‑zero alignment (e.g., check whether future data introduce opposite‑sign curvature that cancels the past contribution).  
   - Try different weighting schemes (e.g., tapered windows, learned weights) rather than a simple uniform average.  
   - Possibly compute the Fisher on a short symmetric window but then apply a correction (e.g., de‑biasing) to account for the fact that future data are not available at decision time.  
   - Run additional diagnostics (e.g., plot the individual Fisher contributions from past vs. future points) to understand the cancellation effect.  
4. The experiment was well‑designed in the sense that it isolated the variable of interest (the type of Fisher smoothing) and measured alignment on a controlled linear‑regression task where the true gradient direction is known. However, the setup may have inadvertently favored the instantaneous Fisher because the true shift direction was derived from the same single point used to compute it, making the alignment trivially high. A more stringent test would use a shift direction computed from a batch of points that does not include the current point, or evaluate on a non‑linear model where the instantaneous Fisher is not a perfect proxy. Thus, while the design was sound for comparing the two smoothing methods, it was biased toward showing the instantaneous Fisher as superior
