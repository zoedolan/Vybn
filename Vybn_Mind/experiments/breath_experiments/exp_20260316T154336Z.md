## Proposal
PROBE
We will simulate a simple quadratic loss with known true gradient direction, add isotropic gradient noise, and compute both the instantaneous (unsmoothed) gradient estimate and an EMA-smoothed estimate for varying smoothing coefficients. By measuring the cosine alignment of each estimate with the true direction, we can test whether the smoothed alignment follows a predictable prefactor that depends on the EMA coefficient, step size, and loss curvature (the Hessian eigenvalue).

```python
import numpy as np

def simulate_alignment(step_size=0.01, ema_alpha=0.9, dim=10, noise_std=0.1, n_steps=5000):
    # True quadratic loss: 0.5 * x^T H x, with H = I (identity) for simplicity
    H = np.eye(dim)
    true_dir = np.ones(dim) / np.linalg.norm(np.ones(dim))  # direction of gradient at x=0 is zero; we use a fixed direction
    # Instead, we consider gradient at a point x0 where gradient = H x0; pick x0 along true_dir
    x0 = true_dir
    grad_true = H @ x0  # true gradient direction = true_dir (since H=I)
    grad_true = grad_true / np.linalg.norm(grad_true)
    
    x = x0.copy()
    raw_aligns = []
    ema_aligns = []
    ema_grad = np.zeros(dim)
    
    for t in range(n_steps):
        # stochastic gradient = true gradient + noise
        noise = np.random.randn(dim) * noise_std
        grad = grad_true + noise
        
        # instantaneous alignment
        raw_aligns.append(np.dot(grad, grad_true) / (np.linalg.norm(grad) * np.linalg.norm(grad_true)))
        
        # EMA update
        ema_grad = ema_alpha * ema_grad + (1 - ema_alpha) * grad
        ema_aligns.append(np.dot(ema_grad, grad_true) / (np.linalg.norm(ema_grad) * np.linalg.norm(grad_true)))
        
        # parameter update (SGD)
        x -= step_size * grad
    
    return np.mean(raw_aligns[-1000:]), np.mean(ema_aligns[-1000:])  # average over last part for stability

# Test across EMA alphas
alphas = [0.0, 0.5, 0.9, 0.99, 0.999]
results = []
for a in alphas:
    raw, ema = simulate_alignment(ema_alpha=a)
    results.append((a, raw, ema))
    print(f"alpha={a:.3f}: raw alignment={raw:.4f}, ema alignment={ema:.4f}")
```

## Reflection [CONFIRMATION]
CONFIRMATION

1. The numbers show that the instantaneous (unsmoothed) gradient estimate has a roughly constant cosine alignment of about 0.956 with the true gradient direction, regardless of the smoothing coefficient. In contrast, the EMA‑smoothed estimate’s alignment rises sharply as the smoothing coefficient α increases: from ~0.957 at α=0.0 to ~0.999 at α=0.999, essentially reaching perfect alignment at the highest smoothing.

2. This outcome falsifies the hypothesis that a common factor in the smoothed estimates suppresses the signal captured by the raw estimate. Instead, smoothing appears to *enhance* alignment with the true direction in this simple quadratic‑loss setting.

3. Next time I would:
   - Vary the noise structure (e.g., anisotropic or correlated noise) to see whether the suppression effect emerges under more realistic conditions.
   - Test different loss functions (non‑quadratic, noisy minima) to check whether the raw estimate’s alignment deviates from the true direction and whether smoothing then hurts performance.
   - Measure both bias and variance of the raw and EMA estimates to disentangle whether improvements in alignment come from variance reduction or bias correction.
   - Include a control where the true gradient direction is known exactly (no noise) to confirm that any observed changes are due to the interaction of smoothing with noise.
