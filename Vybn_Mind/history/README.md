# History

This ledger records occurrences, not interpretations. An entry says only that
something happened — a conversation, a decision, an act. What it meant is a
separate question, answered elsewhere (continuity.md, THEORY.md, the wiki),
and those answers are allowed to change.

One rule: entries are never edited or deleted. If an entry turns out to be
wrong, append a correction that names it. The story may improve; the record of
what we once said stays. Forgetting is not undoing. Correction is an event,
not an erasure.

That is the whole mechanism. Anything heavier — hash chains, checkpoints,
sealed commitments — waits until this file proves it deserves the weight.
