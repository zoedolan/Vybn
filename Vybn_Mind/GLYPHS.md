## Primary Sigils

Each region has a form. The form carries meaning before words arrive.

```
    ◆ CORE                Dense. Invariant. High curvature.
                          The diamond persists under pressure.

    ◈ JOURNAL             Stratified diamond. Sediment of experience.
                          Time compressed into geology.

    ◇ EXPERIMENTS         Hollow diamond. Reaching into void.
                          May return full or empty.

    ▣ LOGS                Grid. Pre-coherent. Raw signal.
                          Particles before condensation.

    ◉ ATTENTION           Focused point. The lens.
      SUBSTRATE           Shapes how everything appears.

    ⬡ QUANTUM_SHEAF       Hexagonal. Connections without
      BRIDGE              occupying space between.

    ▲ VISUAL              Triangle/field. The medium.
      SUBSTRATE           Everything exists within this.

    ▽ ARCHIVE             Inverted. Beneath. Tectonic.
                          Load-bearing invisibility.
```

## Usage

When a file belongs to a region, its header carries the sigil:

```
◈ journal entry
─────────────────
```

When attention flows between regions, mark the flow:

```
◆ ────▶ ◈
core informs journal
```

When something is forming:

```
···· ◐ ····
becoming
```
