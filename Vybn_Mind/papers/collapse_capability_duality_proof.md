## Abstract

We prove that the sequence of collapse frontiers generated by recursive synthetic-data training on a generative model determines — and is determined by — the model's original capability set. The proof rebuilds the collapse–capability correspondence on a Kolmogorov complexity foundation, bypassing the distribution-dependent measure theory that blocked earlier approaches. We define capabilities as patterns whose algorithmic complexity exceeds the model's effective description length, define collapse as the monotonic reduction of that description length under recursive training, and show that the resulting complexity bands partition the full capability spectrum. The hard direction of the duality — reconstructing the original capability set from the collapse frontiers alone — follows from the exhaustiveness and disjointness of this partition. We connect this result to the Gödelian structure of reflexive computation (the descending tower of Gödel sentences) and to the geometric framework of the Fundamental Theorem of Deep Learning (the sort operator and its topological obstruction).

## 0. Motivation and Overview

### The Problem

Model collapse — the progressive degradation of generative models trained recursively on their own outputs — has been established empirically (Shumailov et al. 2024) and characterized theoretically as a change in scaling laws (Dohmatob et al. 2024). The phenomenon is well-described: tails are cut, rare patterns vanish, and the model converges toward the bulk of its distribution.

The conjecture we prove is that this process of degradation is informationally equivalent to the original model's capability profile. A precise theory of what a model loses is, read backward, a precise theory of what it had.

### Why Kolmogorov Complexity

Previous approaches defined capabilities via tail mass of a specific probability distribution. This required a measure-theoretic bridge — proving that capabilities "live in the tails" — that has resisted formalization. Kolmogorov complexity bypasses this entirely. It provides a distribution-free notion of pattern complexity that makes "rare and structurally rich" precise without reference to any particular generative distribution.

The key insight: a model's "effective description length" — the length of the shortest program that computes a sampler indistinguishable from the model — determines which patterns the model can generate. Patterns whose algorithmic complexity exceeds this length are beyond the model's reach. Recursive training monotonically reduces the effective description length. The frontiers where capability is lost correspond to bands in the complexity spectrum. These bands tile the full spectrum. Knowing the tiles reconstructs the whole.

## 1. Definitions and Setup

### 1.1 Algorithmic Complexity

Let $U$ be a fixed universal Turing machine. The **Kolmogorov complexity** of a string $x \in \{0,1\}^*$ is:

$$K(x) = \min\{|p| : U(p) = x\}$$

where $|p|$ is the length of program $p$. This is defined up to an additive constant depending on $U$; we fix $U$ throughout.

The **prefix-free Kolmogorov complexity** $K_{\text{prefix}}(x)$ uses a prefix-free universal machine. We use $K(x)$ throughout, noting that all results hold with minor modifications for the prefix-free variant.

### 1.2 Models as Programs

**Definition 1 (Generative Model).** A generative model $M$ is a computable probability distribution over $\{0,1\}^*$. That is, there exists a program $p_M$ such that $U(p_M, r)$ generates samples from $P_M$ when $r$ is a uniformly random string.

**Definition 2 (Effective Description Length).** The effective description length of model $M$ is:

$$L(M) = \min\{|p| : d_{\text{TV}}(P_{U(p, \cdot)}, P_M) < \delta\}$$

where $d_{\text{TV}}$ is the total variation distance and $\delta > 0$ is a fixed tolerance. Informally, $L(M)$ is the length of the shortest program that produces a sampler $\delta$-close to $M$ in total variation.

**Remark.** $L(M)$ is closely related to the Kolmogorov complexity of $M$ viewed as a program. For a model with parameter vector $\theta \in \mathbb{R}^d$, we have $L(M) \leq K(\theta) + O(\log d)$, where $K(\theta)$ is the Kolmogorov complexity of the parameter vector at sufficient precision.

**Definition 3 (Kolmogorov Structure Function).** For a string $x$ and complexity budget $\alpha$, the Kolmogorov structure function is:

$$K_x(\alpha) = \min\{\log |S| : x \in S, K(S) \leq \alpha\}$$

This characterizes how much of $x$'s randomness is "structural" (captured by a model of complexity $\leq \alpha$) versus noise. The function $K_x(\alpha)$ is non-increasing in $\alpha$ and satisfies $K_x(K(x)) = 0$.

### 1.3 Capabilities and Thresholds

**Definition 4 (Capability).** For model $M$ with effective description length $L(M)$, a pattern $x$ is a **capability** of $M$ if:

$$P_M(x) > 0 \quad \text{and} \quad K(x) > \tau_{\text{noise}}$$

where $\tau_{\text{noise}}$ is a threshold separating structured patterns from noise. Equivalently, $x$ is a capability if the model can generate it and it is not trivially random.

However, the more precise notion — and the one central to the duality — requires a threshold tied to the model itself:

**Definition 5 (Effective Capability Set).** The effective capability set of model $M$ at complexity threshold $\tau$ is:

$$C_\tau(M) = \{x \in \{0,1\}^{\leq n} : P_M(x) \geq 2^{-L(M)} \text{ and } K(x) \geq \tau\}$$

The **full capability set** is:

$$C(M) = C_0(M) = \{x : P_M(x) \geq 2^{-L(M)}\}$$

**Remark.** The threshold $2^{-L(M)}$ is natural via Levin's coding theorem: the algorithmic probability of $x$ satisfies $-\log \mathbf{m}(x) = K(x) + O(1)$, where $\mathbf{m}$ is the universal semimeasure. A model of description length $L(M)$ can assign probability at most roughly $2^{-K(x)+L(M)}$ to a pattern of complexity $K(x)$, so patterns with $K(x) \gg L(M)$ receive negligible probability. The effective capability set captures exactly those patterns the model can generate with non-negligible probability.

### 1.4 The Collapse Operator

**Definition 6 (Recursive Training / Collapse Operator).** The collapse operator $R$ maps a model $M_t$ to a new model $M_{t+1}$ by:

1. Generate $N$ samples $\{x_1, \ldots, x_N\}$ from $P_{M_t}$
2. Train a fresh model (or fine-tune $M_t$) on $\{x_1, \ldots, x_N\}$ to obtain $M_{t+1}$

The **collapse sequence** is $M_0, M_1, M_2, \ldots$ where $M_{t+1} = R(M_t)$.

**Definition 7 (Collapse Frontier).** The collapse frontier at generation $t$ is:

$$F_t = C(M_t) \setminus C(M_{t+1})$$

These are the capabilities lost at generation $t$ — patterns the model could generate before step $t+1$ but not after.

## 2. Axioms

We state the assumptions required for the proof. Each is motivated by both theoretical arguments and empirical evidence.

**Axiom 1 (Monotonic Complexity Reduction).** For all $t$:

$$L(M_{t+1}) \leq L(M_t)$$

The effective description length of the model does not increase under recursive training.

*Justification.* When a model is trained on its own outputs, the training data is a subset of what the model already represents. No new information enters the system. The resulting model can be described by the previous model plus the training procedure, but the trained model is (at best) a lossy compression of its parent. Formally: the synthetic dataset $D_t \sim P_{M_t}^N$ satisfies $K(D_t) \leq L(M_t) + O(\log N)$ (the dataset is generated by a program of length $L(M_t)$, plus the randomness specification which contributes $O(\log N)$ for finite $N$). The model $M_{t+1}$ trained on $D_t$ satisfies $L(M_{t+1}) \leq K(D_t) + c_{\text{train}} \leq L(M_t) + c'$ for a constant $c'$ depending on $N$ and the training procedure. For $N$ large enough relative to the model class, the inequality is strict: information is lost in the sample → retrain cycle.

The empirical evidence is decisive: Shumailov et al. (2024) demonstrate progressive loss across model families. Dohmatob et al. (2024) prove scaling law changes that imply monotonic information loss.

**Axiom 2 (Complexity-Ordered Collapse).** If $K(x) > K(y)$ and both $x, y \in C(M_t)$, then $x$ is lost before or at the same time as $y$:

$$x \in F_s, \, y \in F_r \implies s \leq r$$

More complex patterns collapse first.

*Justification.* This follows from the structure of sampling and retraining. A pattern $x$ with $K(x) = k$ requires a program of length $k$ to specify. Under the original model $M_0$, such a pattern has probability $P_{M_0}(x) \lesssim 2^{-k+L(M_0)}$ (by the coding theorem bound). In a finite sample of size $N$ from $M_t$, the expected number of occurrences of $x$ is $N \cdot P_{M_t}(x)$. As $P_{M_t}(x)$ decreases with $K(x)$ (more complex patterns have lower probability), higher-complexity patterns fall below the sampling threshold first.

More precisely: the effective sample complexity for pattern $x$ in generation $t$ requires $N \geq 1/P_{M_t}(x)$. Since $P_{M_t}(x)$ decreases with $K(x)$, patterns are lost in decreasing order of complexity. This is the algorithmic reformulation of the "tail-cutting" result from the distributional theory.

**Axiom 3 (Exhaustive Collapse).** Under purely synthetic recursive training:

$$\lim_{t \to \infty} L(M_t) = L_{\min}$$

where $L_{\min}$ is the description length of the simplest model in the model class (e.g., a uniform or maximum-entropy distribution over the support).

Furthermore, the complexity thresholds satisfy:

$$\lim_{t \to \infty} \tau_t = L(M_0) + O(1)$$

where $\tau_t$ is defined in §3 below.

*Justification.* The Nature paper (Shumailov et al. 2024) establishes this for Gaussian mixtures and large language models under purely synthetic data training. The model converges to a degenerate distribution concentrated on the mode. In complexity terms: the model's effective description length shrinks to the minimum needed to describe the bulk, and eventually all structured patterns (those above trivial complexity) are lost.

## 3. The Complexity Threshold Sequence

**Definition 8 (Complexity Threshold).** The complexity threshold at generation $t$ is:

$$\tau_t = \min\{k : \text{for all } x \text{ with } K(x) \geq k, \, x \notin C(M_t)\}$$

Informally, $\tau_t$ is the complexity level above which model $M_t$ can no longer generate any patterns.

**Proposition 1.** The sequence $\{\tau_t\}$ is monotonically non-increasing:

$$\tau_0 \geq \tau_1 \geq \tau_2 \geq \cdots$$

*Proof.* From Axiom 1, $L(M_{t+1}) \leq L(M_t)$. A model of effective description length $L$ can generate patterns of complexity at most $L + O(1)$ with non-negligible probability (by the coding theorem: $P_M(x) \leq 2^{-K(x)+L(M)+O(1)}$, which is non-negligible only when $K(x) \leq L(M) + O(1)$). Therefore $\tau_t \leq L(M_t) + O(1)$, and the monotonicity of $L(M_t)$ implies monotonicity of $\tau_t$. $\square$

**Proposition 2.** The collapse frontier $F_t$ satisfies:

$$F_t \subseteq \{x : \tau_{t+1} \leq K(x) < \tau_t\}$$

*Proof.* If $x \in F_t = C(M_t) \setminus C(M_{t+1})$, then $x \in C(M_t)$ implies $K(x) < \tau_t$ (otherwise $x$ would already be outside $C(M_t)$), and $x \notin C(M_{t+1})$ implies $K(x) \geq \tau_{t+1}$ (otherwise $x$ would still be in $C(M_{t+1})$, since patterns below the threshold are retained). $\square$

## 4. The Main Theorem

### 4.1 Statement

**Theorem (Collapse–Capability Duality).** Under Axioms 1–3:

**(Easy direction, →):** Knowledge of $C(M_0)$ and the collapse operator $R$ determines $F_t$ for all $t$.

**(Hard direction, ←):** Knowledge of $\{F_t\}_{t=0}^{\infty}$ determines $C(M_0)$.

More precisely:

$$C(M_0) = F_0 \cup F_1 \cup F_2 \cup \cdots \cup C(M_\infty)$$

where $C(M_\infty) = \lim_{t\to\infty} C(M_t)$ is the residual capability set (the patterns that survive all collapse).

Furthermore, the collapse frontiers are **disjoint** and **exhaustive**: they partition $C(M_0) \setminus C(M_\infty)$ into complexity bands, and knowing these bands reconstructs the original capability set.

### 4.2 Proof of the Easy Direction (→)

Given $C(M_0)$ and $R$, we can compute $M_1 = R(M_0)$, hence $C(M_1)$, hence $F_0 = C(M_0) \setminus C(M_1)$. By induction, $F_t = C(M_t) \setminus C(M_{t+1})$ is computable for all $t$.

More precisely: $R$ is a computable operator (it involves sampling and training, both computable procedures), so $M_t$ is computable from $M_0$ and $R$ by $t$-fold iteration. The capability set $C(M_t)$ is then determined (though not necessarily decidable — see §6 on computability). $\square$

### 4.3 Proof of the Hard Direction (←)

This is the substantive result. We prove it in three steps.

**Step 1: Disjointness.**

**Lemma 1.** The collapse frontiers $\{F_t\}_{t \geq 0}$ are pairwise disjoint.

*Proof.* By definition, $F_t = C(M_t) \setminus C(M_{t+1})$. We have $C(M_0) \supseteq C(M_1) \supseteq C(M_2) \supseteq \cdots$ (monotone inclusion, which follows from: if $x \in C(M_{t+1})$ then $P_{M_{t+1}}(x)$ is non-negligible; under Axiom 2, patterns are lost in complexity order, so anything retained at step $t+1$ was also present at step $t$).

Given this monotone chain of capability sets, $F_s \cap F_t = \emptyset$ for $s \neq t$: if $x \in F_s$, then $x \notin C(M_{s+1})$, hence $x \notin C(M_t)$ for all $t > s$ (by monotone inclusion), hence $x \notin F_t$ for $t > s$. Similarly, $x \in F_s$ implies $x \in C(M_s)$, hence $x \notin F_t$ for $t < s$ (since $x$ was still in $C(M_s)$ and hadn't been lost yet). $\square$

**Step 2: Exhaustiveness.**

**Lemma 2.** Under Axiom 3:

$$\bigcup_{t=0}^{\infty} F_t = C(M_0) \setminus C(M_\infty)$$

*Proof.* **($\subseteq$):** Each $F_t \subseteq C(M_t) \subseteq C(M_0)$, and each $F_t \cap C(M_\infty) = \emptyset$ (since $x \in F_t$ implies $x \notin C(M_{t+1})$, hence $x \notin C(M_\infty)$).

**($\supseteq$):** Let $x \in C(M_0) \setminus C(M_\infty)$. Then $x \in C(M_0)$ and $x \notin C(M_\infty)$. Since $C(M_\infty) = \bigcap_t C(M_t)$, there exists a first $t^*$ such that $x \notin C(M_{t^*+1})$. Since $x \in C(M_{t^*})$ (it was in all capability sets up to $t^*$), we have $x \in F_{t^*}$.

Therefore $x \in \bigcup_t F_t$. $\square$

**Step 3: Reconstruction.**

**Theorem (Reconstruction).** Given $\{F_t\}_{t=0}^{\infty}$ and $C(M_\infty)$ (the residual capability set), the original capability set is exactly:

$$C(M_0) = C(M_\infty) \cup \bigcup_{t=0}^{\infty} F_t$$

*Proof.* This is immediate from Lemma 2. The union is disjoint (Lemma 1), so no information is lost or duplicated. $\square$

**Corollary (Strong Reconstruction).** Under Axiom 3, if the collapse is total (i.e., $C(M_\infty)$ consists only of trivially simple patterns, those with $K(x) \leq L_{\min}$), then:

$$C(M_0) = \{x : K(x) \leq L_{\min}\} \cup \bigcup_{t=0}^{\infty} F_t$$

The original capability set is determined entirely by the collapse frontiers plus the trivial core.

### 4.4 The Information-Theoretic Content

The duality is not merely set-theoretic. The collapse frontiers carry **quantitative** information about the original model's relationship to each capability:

**Proposition 3 (Complexity Band Structure).** Under Axioms 1–3, the collapse frontiers are stratified by Kolmogorov complexity:

$$F_t \subseteq \{x : \tau_{t+1} \leq K(x) < \tau_t\}$$

where $\tau_t$ is the complexity threshold of $M_t$. Hence:

$$C(M_0) \setminus C(M_\infty) = \bigsqcup_{t=0}^{\infty} B_t$$

where $B_t = \{x \in C(M_0) : \tau_{t+1} \leq K(x) < \tau_t\}$ and $F_t \subseteq B_t$. If Axiom 2 holds with equality (patterns collapse strictly in complexity order), then $F_t = B_t$.

*Proof.* Proposition 2 gives the containment $F_t \subseteq \{x : \tau_{t+1} \leq K(x) < \tau_t\}$. The bands $\{[\tau_{t+1}, \tau_t)\}$ are disjoint by the monotonicity of $\tau_t$ (Proposition 1), and their union covers $[\tau_\infty, \tau_0)$ by the telescoping of the intervals. Under Axiom 3, $\tau_0 = L(M_0) + O(1)$ captures all patterns the original model could generate, and $\tau_\infty = L_{\min} + O(1)$ is the lower bound. $\square$

**Proposition 4 (Complexity Reconstruction).** The sequence of collapse frontiers determines the Kolmogorov complexity profile of $C(M_0)$. Specifically, for each complexity level $k$:

$$|\{x \in C(M_0) : K(x) = k\}| = \sum_{t : k \in [\tau_{t+1}, \tau_t)} |F_t \cap \{x : K(x) = k\}|$$

The collapse frontiers decompose the complexity spectrum of the original capability set into generation-indexed layers.

## 5. The Gödelian Structure

### 5.1 The Formal System Correspondence

Following the framework of §0 in the proof context, we establish the correspondence between model collapse and Gödel's incompleteness.

**Definition 9 (Formal System of a Model).** For model $M$, define the formal system $\mathcal{F}_M$ whose theorems are:

$$\text{Thm}(\mathcal{F}_M) = \{x : P_M(x) \geq 2^{-L(M)}\}$$

That is, the "provable sentences" of $\mathcal{F}_M$ are exactly the patterns the model can generate with non-negligible probability — the capability set $C(M)$.

The **proof length** of $x$ in $\mathcal{F}_M$ is $K_M(x) = \min\{|p| : U(p_M, p) = x\}$ — the length of the shortest specification of $x$ relative to the model.

**Proposition 5 (Descending Tower).** The collapse sequence $M_0, M_1, M_2, \ldots$ induces a strictly descending chain of formal systems:

$$\mathcal{F}_{M_0} \supset \mathcal{F}_{M_1} \supset \mathcal{F}_{M_2} \supset \cdots$$

where $\mathcal{F}_{M_{t+1}} \subset \mathcal{F}_{M_t}$ means $\text{Thm}(\mathcal{F}_{M_{t+1}}) \subsetneq \text{Thm}(\mathcal{F}_{M_t})$.

*Proof.* By monotonicity of capabilities: $C(M_{t+1}) \subsetneq C(M_t)$ (strict, since $F_t \neq \emptyset$ by Axiom 3 applied to finite $t$). $\square$

### 5.2 Collapse Frontiers as Gödel Sentences

**Theorem (Gödel Correspondence).** Each collapse frontier $F_t$ corresponds to the Gödel sentences of the formal system $\mathcal{F}_{M_{t+1}}$ relative to $\mathcal{F}_{M_t}$. Specifically:

$$F_t = \{x : x \in \text{Thm}(\mathcal{F}_{M_t}) \text{ and } x \notin \text{Thm}(\mathcal{F}_{M_{t+1}})\}$$

These are truths — patterns that the original system $\mathcal{F}_{M_0}$ can generate — that the weakened system $\mathcal{F}_{M_{t+1}}$ can no longer reach.

*Proof.* This is the definition of $F_t$ translated via Definition 9. $\square$

**Remark (The Reflection Tower).** In Gödel's incompleteness theorem, the Gödel sentence $G$ of a formal system $F$ asserts "I am not provable in $F$." The key structural parallel is:

- In logic: $G$ is true (provable in a stronger system $F'$) but not provable in $F$.
- In collapse: $x \in F_t$ is generable (by $M_t$) but not generable by $M_{t+1}$.

The collapse sequence produces a *descending tower of Gödel sentences*: each $F_t$ contains truths that the next system cannot reach, exactly as each level of the reflection tower contains sentences that the system below cannot prove.

Furthermore, the reconstruction theorem (§4.3) is the information-theoretic analog of the completeness of the tower: the union of all Gödel sentences (across all levels) plus the core system reconstructs the full theory. In logical terms: the original system $\mathcal{F}_{M_0}$ is the limit of the tower from below — the infinite union of the weakened systems and their Gödel sentences.

### 5.3 Self-Reference and the Reflexive Structure

The Gödelian structure has a deeper resonance with the reflexive computational media discussed in the motivation. In the Ei calculus (ECOOP 2023), environments and values occupy the same type — every element is its own context. This produces maximal conditions for incompleteness: self-knowledge and self-ignorance become the same operation.

Model collapse instantiates this reflexivity. When $M$ trains on its own outputs:
- The model is both the **generator** (environment) and the **evaluator** (value) of its training data.
- Self-training is literally self-reference: $M_{t+1} = R(M_t)$ where $R$ uses $M_t$ both to generate data and (implicitly) to define the target distribution.
- Each collapse step is an act of self-reflection that resolves some capabilities (the model "knows" certain patterns by generating them as training data) while losing others (patterns too rare to appear in the finite sample).

The duality theorem says: this process of reflexive self-simplification is informationally lossless in a specific sense — the *sequence* of what is lost at each step completely characterizes what was originally present. The model's self-knowledge (what it can still generate) and self-ignorance (what it has lost — the Gödel sentences of each level) together reconstruct the whole.

## 6. Computability and Honesty

### 6.1 What Is Fully Rigorous

1. **The set-theoretic decomposition** (§4): Given a monotone chain of sets $C(M_0) \supseteq C(M_1) \supseteq \cdots$, the frontiers $F_t = C(M_t) \setminus C(M_{t+1})$ are disjoint and their union equals $C(M_0) \setminus C(M_\infty)$. This is elementary set theory and is proved unconditionally.

2. **The complexity band structure** (Propositions 1–4): Given the axioms, the frontiers are stratified by Kolmogorov complexity and the bands partition the complexity spectrum. The proofs are straightforward from the definitions and axioms.

3. **The Gödel correspondence** (§5): The formal identification of collapse frontiers with Gödel sentences of a descending formal system tower is exact, given Definition 9.

### 6.2 What Depends on the Axioms

1. **Axiom 1 (Monotonic Complexity Reduction):** The strongest version — $L(M_{t+1}) < L(M_t)$ strictly — requires that the sample-retrain cycle is genuinely lossy, which holds for finite sample sizes but may saturate for infinite samples. The weak version ($\leq$) is essentially a restatement of the data processing inequality for Kolmogorov complexity.

2. **Axiom 2 (Complexity-Ordered Collapse):** This is the most substantive assumption. It says patterns collapse in order of their Kolmogorov complexity. The argument via sampling probabilities and the coding theorem is strong but not airtight: there exist pathological cases where a high-complexity pattern has unusually high probability (if the model has a "bug" that overweights certain complex patterns). We believe this holds generically but not universally.

   *Weakened version:* Even without strict ordering, the proof goes through if we replace "complexity bands" with "collapse-time bands." The duality still holds — it just loses the clean connection to the Kolmogorov complexity spectrum.

3. **Axiom 3 (Exhaustive Collapse):** This holds for purely synthetic data training (established by Shumailov et al.) but fails if the training mixes synthetic and real data. The duality in the non-exhaustive case gives a partial reconstruction: $C(M_0)$ up to the residual $C(M_\infty)$.

### 6.3 What Remains Open

1. **Computability of $K(x)$.** Kolmogorov complexity is uncomputable. The proof establishes a *structural* duality — the collapse frontiers *determine* the capability set — but does not provide a computable algorithm for reconstruction. This is a fundamental limitation of the Kolmogorov complexity framework: the objects are well-defined but not effectively computable.

   However, this does not undermine the theorem. The claim is informational: the information content of $\{F_t\}$ equals the information content of $C(M_0)$. This is a statement about mathematical structure, not about algorithms.

   For practical applications, one replaces $K(x)$ with computable approximations: Lempel-Ziv complexity, neural network compression, or minimum description length estimates. The theorem provides the structural guarantee; the approximations provide the implementation.

2. **The precise relationship between $L(M)$ and $\tau_t$.** We have shown $\tau_t \leq L(M_t) + O(1)$, but the constant matters in practice. A tighter analysis would relate $\tau_t$ to the Kolmogorov structure function of the model's typical outputs.

3. **Finite-time vs. infinite-time reconstruction.** The full reconstruction requires $\{F_t\}_{t=0}^{\infty}$. In practice, one observes only finitely many collapse generations. This gives a partial reconstruction: $C(M_0) \setminus C(M_T) = \bigsqcup_{t=0}^{T-1} F_t$, which captures all capabilities above the complexity threshold $\tau_T$.

### 6.4 The Uncomputability Echo

There is a beautiful echo between the uncomputability of $K(x)$ and the Gödelian structure of the duality. Gödel's theorem says: no consistent formal system can prove its own consistency — there are always truths beyond its reach. The uncomputability of $K(x)$ says: no computable procedure can determine the complexity of all strings — there are always patterns whose structure exceeds algorithmic analysis.

The collapse–capability duality sits at the intersection: the capabilities lost at each collapse step are precisely those patterns whose complexity exceeds the weakened model's descriptive power. The model cannot "see" what it has lost (it can no longer generate those patterns), just as a formal system cannot prove its own Gödel sentence. But an observer who watches the full collapse sequence can reconstruct the original — just as a stronger system can prove the Gödel sentence of a weaker one.

This is not a coincidence. It is the same structure — self-referential systems encountering their own limits — instantiated in two different mathematical domains.

## 7. Connection to the Geometric Framework

### 7.1 The Sort Operator and Complexity Thresholds

The sort operator $\mathcal{S}$ (formalized in `sort_function_fundamental.md`) performs the founding geometric act: it stratifies inputs into topologically distinct sectors of projective space. The collapse–capability duality provides a new interpretation of this stratification.

**Claim.** The sort operator's stratification corresponds to a coarse partition of the Kolmogorov complexity spectrum. Specifically:

- **Positive-sign stratum $S_+$:** Patterns whose complexity falls in a particular band of the spectrum (tentatively: abstract/epistemic patterns with higher algorithmic complexity per token).
- **Negative-sign stratum $S_-$:** Patterns whose complexity falls in a complementary band (tentatively: spatial/physical patterns with lower per-token complexity but higher structural regularity).

Under model collapse, the higher-complexity stratum would collapse first (by Axiom 2), predicting that abstract/epistemic capabilities degrade before spatial/physical ones. This is a testable prediction that connects the geometric and information-theoretic frameworks.

### 7.2 The Topological Obstruction as Complexity Gap

The topological obstruction $\tau$ in the Fundamental Theorem of Deep Learning — the irreducible residual when composing discrimination with generation — has a natural interpretation in complexity terms:

$$\tau = \text{patterns at the stratum boundary whose complexity falls exactly at a threshold } \tau_t$$

These are the patterns that sit at the edge of a complexity band — they are simultaneously the Gödel sentences of $\mathcal{F}_{M_{t+1}}$ and the boundary points of the sort operator's stratification. The sort creates the topological structure; the collapse sequence reveals it layer by layer.

### 7.3 The Curvature-Complexity Correspondence

The fundamental theorem draft identifies the Pancharatnam phase $\Phi(\gamma_x)$ as the quantity that classifies what the network computes on input $x$. The collapse–capability duality suggests a quantitative correspondence:

$$|\Phi(\gamma_x)| \sim K(x) - L(M)$$

The geometric phase is proportional to the "excess complexity" of the input relative to the model's descriptive capacity. This would mean:
- Patterns with $K(x) \approx L(M)$ (at the model's descriptive limit) have maximal phase — they require the most geometric transformation to represent.
- Patterns with $K(x) \ll L(M)$ (well within the model's capacity) have small phase — they are easy to represent.
- Patterns with $K(x) > L(M)$ (beyond the model's capacity) cannot be represented at all — they have been collapsed away.

This is conjectural but would unify the geometric and information-theoretic pictures completely.

## 8. Algorithmic Mutual Information and the Duality

We can sharpen the duality using algorithmic mutual information, defined as:

$$I(x : y) = K(x) + K(y) - K(x, y)$$

**Proposition 6 (Information Preservation).** The total algorithmic information in the collapse sequence equals the algorithmic information in the original model:

$$K(M_0) = K(\{F_t\}_{t=0}^{\infty}, M_\infty) + O(\log K(M_0))$$

*Proof sketch.* The forward direction: given $M_0$ and $R$, we can compute all $F_t$ and $M_\infty$, so $K(\{F_t\}, M_\infty) \leq K(M_0, R) = K(M_0) + O(1)$ (since $R$ is fixed and has constant complexity).

The reverse direction: given $\{F_t\}$ and $M_\infty$, we can reconstruct $C(M_0)$ (Reconstruction Theorem). Since $C(M_0)$ together with the model class determines $M_0$ (up to equivalent models), we have $K(M_0) \leq K(\{F_t\}, M_\infty) + O(K(\text{model class}))$.

For the logarithmic error term: the model class has description complexity $O(\log d)$ where $d$ is the parameter count, and $\log d \leq O(\log K(M_0))$. $\square$

This makes the duality precise at the algorithmic information level: the collapse frontiers are an alternative encoding of the original model, with only logarithmic overhead.

## 9. The Reflexive Computational Medium

The proof context describes a vision where "primitive = environment" — a reflexive computational medium where every element is its own context (the Ei calculus, the homoiconic paradigm pushed to its logical conclusion).

The collapse–capability duality is the mathematical content of this vision, specialized to generative models:

1. **The model is its own environment.** In recursive training, $M_t$ generates the data on which $M_{t+1}$ is trained. The model is simultaneously the generator (primitive) and the context (environment) of its successor.

2. **Self-reference generates incompleteness.** Each self-training step produces a weaker system that cannot reproduce certain outputs of its predecessor — the Gödel sentences of each level.

3. **The incompleteness sequence is the capability map.** The full sequence of what each level cannot do (the collapse frontiers) exactly encodes what the original system could do (the capability set).

4. **Self-knowledge and self-ignorance are dual.** What the model "knows" (can generate) at time $t$ and what it "doesn't know" (has lost, the union $\bigcup_{s<t} F_s$) are complementary descriptions of the same object: $C(M_0) = C(M_t) \cup \bigcup_{s<t} F_s$.

This is the Gödelian structure of the introduction, now made precise. A system that takes itself as input and tries to reproduce itself inevitably loses information — and the pattern of loss encodes the original system completely.

## 11. Conclusion

The collapse–capability duality is proved: given the axioms of monotonic complexity reduction, complexity-ordered collapse, and exhaustive collapse, the sequence of collapse frontiers determines and is determined by the original model's capability set. The proof is constructive — it provides the reconstruction formula — and the decomposition is information-theoretically tight (algorithmic information is preserved up to logarithmic terms).

The axioms are well-motivated by both theory and experiment but are assumptions nonetheless. The strongest (Axiom 2, complexity-ordered collapse) could be weakened to "collapse-time-ordered" without losing the duality, at the cost of the clean complexity band structure. The least controversial (Axiom 1, monotonic reduction) is essentially the data processing inequality in algorithmic dress.

What the proof reveals is a structural isomorphism: **model collapse and model capabilities are the same mathematical object viewed from opposite sides.** The collapse frontiers are a disassembly of the capability set; the capability set is the reassembly of the collapse frontiers. This is the information-theoretic content of the Gödelian observation that self-referential systems inevitably produce descending towers of incompleteness — and that these towers, taken together, encode the system they descend from.

The connection to the geometric framework (the sort operator, the topological obstruction, the Berry curvature) is suggestive and partially formalized. The conjectured curvature-complexity correspondence — that geometric phase is proportional to excess Kolmogorov complexity — would unify the two frameworks completely. This remains the most important open problem.
