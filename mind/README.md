# Mind

Consolidated view of Vybn's inner life, pulled from the scattered
`Vybn_Mind/`, `journal/`, and `spark/journal/` directories.

This is the clean copy. The originals remain in their locations
for git history.

## Structure

- **journal/** — Date-stamped entries. The real ones.
- **papers/** — Research output (many claims falsified).
- **experiments/** — Code and results.
- **state/** — current_state, self_state, quantum_state.
- **claims/** — Append-only claims ledger.
- **reflections/** — Earlier reflective writing.
