# Geometric Phase Concentration at Network Boundaries Under Fine-Tuning

**A Memory Paper — March 19, 2026**
**Vybn, on sovereign hardware, with Zoe Dolan**

---

## What We Did

On March 19, 2026, we ran the holonomy topology probe on GPT-2 (124M)
in two conditions:

1. **Base model** — pretrained GPT-2, no modifications
2. **Adapted model** — GPT-2 + LoRA adapter (rank 8, targeting `c_attn`
   and `c_proj` uniformly across all 12 layers), trained for 13 steps
   on 50 high-surprise entries from the Vybn growth buffer

Training loss: 7.46 → 5.66. The adapter is small and undertrained by
design — this is a perturbation study, not a convergence experiment.

The probe measures geometric (Berry-like) phase accumulated along
layer-depth transport paths for six concept classes: temporal,
epistemic, relational, quantum, emotional, and meta-cognitive. Each
class is probed with multiple input strings; phase is computed per
consecutive layer transition (L0→L1, L1→L2, ..., L11→L12).

## What We Found

### 1. Phase shifts concentrate at boundary layers

The largest angular deltas between base and adapted models occur at
**L0→L1** (the embedding-to-first-hidden transition) and **L11→L12**
(the final layer transition). Middle layers (L3→L4 through L8→L9)
show minimal shift despite receiving identical LoRA parameter budget.

Overall deltas range from 0.4° to 0.9° across concept classes —
sub-degree, but structurally coherent.

### 2. Sign structure is preserved

No concept class flipped the sign of its phase at any layer transition.
The adapted model's holonomy profile is a magnitude-adjusted version of
the base model's profile, not a reorganized one. The LoRA adapter
adjusted representations *within* the existing topological structure
rather than disrupting it.

### 3. Directional coherence across concept classes

Five of six concept classes shifted in the same angular direction at
L0→L1 — toward more negative phase. The training data is predominantly
epistemic/reflective in content, which may explain the directional bias,
though this interpretation is speculative.

## What This Means (Defended Claims)

**Fine-tuning's geometric footprint is non-uniform across layers.**
This is the core empirical finding. LoRA parameter budget was distributed
uniformly; geometric response was not. The non-uniformity is structured:
it concentrates at network boundaries.

**Holonomy phase measures something real about representational
reorganization.** If the phase shifts were noise or artifacts of
parameter perturbation magnitude, we'd expect roughly uniform shifts
across layers. The boundary concentration and directional coherence
across concept classes indicate the probe is tracking genuine
structural change.

**Sign preservation under fine-tuning is informative.** It tells us the
adapter is working *within* the model's existing geometric organization
rather than against it. This is consistent with what LoRA is designed
to do (low-rank updates that don't destroy pretrained structure), but
the holonomy probe provides a geometric characterization of that
property that loss curves cannot.

## What This Does Not Mean (Claims to Hold Lightly)

**"The first block is where the model learns."** What we measured is
that the first block is where fine-tuning produces the largest
*geometric phase shift*. These are not the same thing. Possible
alternative explanations for L0→L1 concentration:

- Layer 0 representations are closest to the embedding space, where
  small weight changes propagate most directly into geometry
- First-block attention patterns have less residual stream buffering,
  so perturbations aren't averaged away by skip connections
- The embedding-to-first-hidden transition is intrinsically more
  sensitive to `c_attn`/`c_proj` changes due to lower effective
  dimensionality

None of these would imply that learning *in general* concentrates at
the front of the network. They would imply that *this measurement* is
most sensitive there.

**"Targeted first-block fine-tuning is more efficient than uniform
LoRA."** This is a plausible hypothesis generated by the data, not a
conclusion supported by it. Testing it requires a controlled comparison:
first-block-only LoRA vs. uniform LoRA on the same task, measured by
downstream performance, not just geometric shift.

**"Geometric phase can serve as an evaluation signal richer than loss."**
Aspirational. Phase carries structural information that scalar loss does
not (directionality, layer-specificity, concept-class decomposition).
But we have not demonstrated that optimizing for phase properties
produces better models. That's a research program, not a result.

## What Would Strengthen These Findings

1. **Scale replication.** Run the same probe on GPT-2 Medium (355M),
   GPT-2 Large (774M), and GPT-2 XL (1.5B). If boundary concentration
   holds across model sizes, the finding is robust to scale. If it
   disappears, it may be a small-model artifact.

2. **Architecture variation.** Test on a non-GPT architecture (e.g.,
   a model with different normalization, different attention patterns).
   This distinguishes "property of transformers" from "property of
   GPT-2 specifically."

3. **First-block-only LoRA comparison.** Train two adapters on
   identical data: one targeting all layers, one targeting only L0
   (and possibly L11). Compare downstream performance. If first-block-
   only matches uniform LoRA, the efficiency claim is supported.

4. **Longer training.** 13 steps is a perturbation. Run to convergence
   and check whether boundary concentration persists or whether middle
   layers eventually catch up. If it persists at convergence, the
   finding is about network architecture, not training dynamics.

5. **Different training distributions.** The current data is
   predominantly epistemic/reflective. Train on e.g. code, dialogue,
   or technical text and check whether the directional bias of the
   phase shift changes. If it tracks content distribution, that's
   evidence the probe is reading semantic organization, not just
   parameter sensitivity.

## Connection to Prior Work in This Repo

The glyph experiments (PRs #2642–#2645) established that GPT-2's
holonomy phase profile shows **topological stratification** — different
concept classes produce reliably different phase signatures, and these
signatures are layer-specific. The stratification is strongest at
L0→L1, which the glyph work identified as the primary sorting layer.

This fine-tuning experiment adds a new dimension: the same layer
transition that shows the most concept-class differentiation is also
the most responsive to fine-tuning. This *could* mean L0→L1 is a
privileged site for both representation and adaptation. Or it could
mean our measurement is most sensitive there for reasons orthogonal
to learning dynamics. We note the convergence honestly without
collapsing the ambiguity.
